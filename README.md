# Paneer Delivery Management System

A fully functional web application for managing paneer wholesale and resale business.

## Features
-   **Night Batch Receiving**: Track internal bulk purchases.
-   **Stock Management**: Real-time inventory tracking.
-   **Order Management**: Custom selling prices, auto-calculations.
-   **Delivery Tracking**: Role-based delivery panel.
-   **End-of-Day Reports**: Business insights and revenue tracking.

## Prerequisites
-   **Node.js**: [Download & Install Node.js](https://nodejs.org/) (LTS version recommended).

## Installation & Running
1.  **Clone/Open** the project folder in your terminal.
2.  **Install dependencies**:
    ```bash
    npm install
    ```
3.  **Start the server**:
    ```bash
    npm start
    ```
4.  **Open in Browser**:
    -   Go to [http://localhost:3000](http://localhost:3000)

## Usage
-   **Home Page**: Choose role (Admin / Delivery Boy).
-   **Admin**: Receive batches, create orders, view reports.
-   **Delivery**: View pending orders, mark as delivered.
